# Cabinet Dentaire - Site Web

## À propos du projet

Site web moderne pour un cabinet dentaire offrant une présentation professionnelle des services et facilitant la prise de rendez-vous.

## Technologies utilisées

Ce projet est construit avec :

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Installation et développement

### Prérequis

- Node.js (installation recommandée via [nvm](https://github.com/nvm-sh/nvm#installing-and-updating))
- npm ou bun

### Installation

```sh
# Cloner le dépôt
git clone <URL_DU_DEPOT>

# Naviguer dans le répertoire
cd denti-site-magic

# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev
```

## Scripts disponibles

- `npm run dev` - Démarre le serveur de développement
- `npm run build` - Crée la version de production
- `npm run build:dev` - Crée une version de développement
- `npm run lint` - Vérifie le code avec ESLint
- `npm run preview` - Prévisualise la version de production

## Déploiement

Le projet peut être déployé sur n'importe quelle plateforme supportant les applications Vite (Vercel, Netlify, etc.).
